## Setup and Installation

Clone the Repo.

```bash
git clone https://github.com/kumudgupta76/text-collector.git

cd text-collector

./gradlew bootRun

# or
./gradlew build
java -jar build/libs/text-collector-0.0.1-SNAPSHOT.jar 

```

You can see results on [this](http://localhost:8080/home)
